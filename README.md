# Weather API App

A weather application built with HTML, CSS, and JavaScript that fetches real-time weather data from an external API.

## Features
- Search weather by city  
- Fetch real-time weather data from API  
- Display temperature and weather conditions  
- Clean and responsive UI  

## Technologies
- HTML  
- CSS  
- JavaScript  
- REST API (OpenWeather API or your API name)
